/////////      JS - ANGULAR      /////////
////////////      instalar    ////////////

----------------------------------------
versão do Node e NPM
----------------------------------------

        node -v
        npm -v


----------------------------------------
instalar o Angular
----------------------------------------

        sudo npm install -g @angular/cli

        g - global 
            significa computador inteiro


----------------------------------------
verificar a versão do Angular
----------------------------------------

        ng v

----------------------------------------
criar uma área de trabalho - workspace
e tambem uma nova aplicação
----------------------------------------

        ng new <nome da aplicação> 


----------------------------------------
entre dentro da pasta da aplicação 
e abrir ela no navegador
----------------------------------------

        ng serve --open 




ou somente

        ng serve 




e ir até o navegador e digitar

        http://localhost:4200/
